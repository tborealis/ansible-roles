# new_relic

Configures New Relic PHP agent for application monitoring.

## Role Variables

| Variable | Default | Description |
|----------|---------|-------------|
| `newrelic_enable` | `false` | Enable New Relic agent |
| `newrelic_browser_auto_instrument` | `false` | Enable browser auto-instrumentation |
| `newrelic_distributed_tracing_enabled` | `true` | Enable distributed tracing |
